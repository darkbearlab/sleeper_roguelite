# SLEEPER 地圖待辦清單（依 archetype）

> 用途：在 `editor.html` 做地圖時的待辦＋規格速查。做完一張就打勾。
> 來源＝主程式 `index.html` 的 `MAP_ARCHETYPES`（11 型）。**改 index 的清單時記得同步本檔。**

## 怎麼用（編輯器流程）
1. 開 `editor.html` →「本圖屬性」下拉設三個標籤：
   - **類型（category）**＝下面 11 型之一（吃 `MAP_ARCHETYPES`）。
   - **難度（tier）**＝ 1 / 2 / 3（roadmap 依進度抽：前段只抽 tier1、中段 ≤2、後段 ≤3）。
   - **用途（roles）**＝勾 `normal`（一般戰鬥池）/ `boss`（壓軸池）。兩個都不勾＝不會被 roadmap 自動抽到。
2. 存成 `maps.json`（editor 的下載整份）。
3. 把 `maps.json` 丟給 Claude → bind 進主遊戲 `MAPS`（順手補 enemySpawns / caches 等）。

**規則提醒**
- 戰鬥圖才標 `roles`；事件專用圖（`eventMap:true`）不進 roadmap 隨機池（像現在的暴動/金庫）。
- 別讓「擋移動的牆/區域」壓到 waypoint 連線（editor 會紅標 `edgeBlocked`），否則 AI 會卡死。
- 同類型（category）盡量別只做一張＝roadmap 會避免連續同類，多張才有變化。

## 現況總覽
- **已在抽選池**：open、facility、corridor、outpost
- **只在事件圖（未進池）**：urban（暴動街區）、vault（金庫）

## 清單（11 型）

### 1. open — 開闊庭院
- 用途：開闊長視線、零散掩體；走位 / overwatch / 繞背
- 建議：tier 1–2，roles `normal`
- 現況：✅ 已有 `alpha`（tier1）
- [ ] 補一張 tier2 變體（更多掩體島 / 不同動線）

### 2. facility — 門禁設施
- 用途：房間＋門，破門清屋、門控視野
- 建議：tier 2–3，roles `normal` +（可）`boss`
- 現況：✅ `bravo`（tier2, normal+boss）、✅ `finale`（tier3, boss）
- [ ] 補一張 tier3 `normal`（讓後段一般戰也有高難設施圖）

### 3. corridor — 雙線哨廊
- 用途：平行雙線，換道側翼 / 潛行繞行
- 建議：tier 1–2，roles `normal`
- 現況：✅ `lanes`（tier1）
- [ ] 補一張 tier2 變體

### 4. outpost — 前哨站
- 用途：哨點＋混合結構，潛行通過警戒線
- 建議：tier 2，roles `normal` + `boss`
- 現況：✅ `outpost`（tier2, normal+boss）
- [ ] （可選）tier1 簡化版當前段圖

### 5. urban — 街區
- 用途：街道＋建築，巷戰 / 多勢力交火
- 建議：tier 2–3，roles `normal`
- 現況：⚠️ 只有事件圖 `riot`（eventMap，不進池）
- [ ] 做一張可進「一般池」的街區戰鬥圖（tier2）
- [ ] （可選）tier3 大型街區

### 6. vault — 金庫 / 保險庫
- 用途：目標房＋門鎖開關，搜打撤式 heist
- 建議：tier 2–3，roles `normal` /（可）`boss`
- 現況：⚠️ 只有事件圖 `vault`（eventMap，不進池）
- [ ] 做一張可進池的金庫圖（或把現有金庫加 roles 進池）

### 7. cqb — 狹巷 / CQB
- 用途：貼身近戰，霰彈 / 小刀吃香、視線極短
- 建議：tier 1–2，roles `normal`
- 現況：❌ 未做
- [ ] tier1 小型 CQB
- [ ] tier2 多房間 CQB

### 8. warehouse — 倉儲
- 用途：大型室內＋貨架掩體走廊，物資密集
- 建議：tier 2–3，roles `normal` +（可）`boss`
- 現況：❌ 未做
- [ ] tier2 倉儲（貨架排＝低矮牆/掩體走廊）

### 9. field — 郊野
- 用途：超開闊＋草叢遮蔽，長程 / DMR / 狙擊
- 建議：tier 1–3，roles `normal`
- 現況：❌ 未做（註：搜打撤 arcade 的大圖 `bigyard` 是 field 類，但屬 arcade 分支、不在主遊戲池）
- [ ] tier2 郊野（大量草叢 + 長視線）

### 10. compound — 聚落 / 院落
- 用途：多棟建築＋中庭，空間分割、攻防轉換
- 建議：tier 2–3，roles `normal` + `boss`
- 現況：❌ 未做
- [ ] tier2 院落
- [ ] tier3 大型聚落（可當 boss 圖）

### 11. choke — 隘口 / 橋
- 用途：單一強制咽喉，硬突破 / 伏擊
- 建議：tier 2–3，roles `normal` /（可）`boss`
- 現況：❌ 未做
- [ ] tier2 隘口 / 橋
